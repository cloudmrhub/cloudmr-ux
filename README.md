# cloudmr-ux

## for dev:
clone this package
```bash
cd cloudmr-ux
```
build it
```
npm install
npm run build
```
in a different directory create a test app
```bash
npx create-react-app local                    
```

install cloudmr-ux (locally)
```bash
npm i /path/to/the/package/cloudmr-ux

```
that's it!
