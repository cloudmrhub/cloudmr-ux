# cloudmr-ux

## for dev:
1. clone this package
    ```bash
    cd cloudmr-ux
    ```
1. create a pakage global link

    ```bash
    npm link
    ```
1. in a different directory run 
    ```bash
    npx create-react-app local                    
    ```
1. install cloudmr-ux (locally)
    ```bash
    npm i /path/to/the/package/cloudmr-ux

    ```
1. taht's it!
